# Style Guide v1

## Visual Direction
- Theme: luminous white hardware with brushed-gold accents
- Mood: premium studio outboard, vocal-focused, expensive but restrained
- Contrast strategy: dark warm typography on bright surfaces for DAW visibility

## Color Palette
- `--bg-main`: `#F3EFE6`
- `--bg-panel`: `#FFF9F1`
- `--bg-shadow`: `rgba(76, 52, 24, 0.12)`
- `--text-primary`: `#20160D`
- `--text-secondary`: `#6C5844`
- `--accent-gold`: `#B98934`
- `--accent-gold-soft`: `#E3C27A`
- `--accent-outline`: `#D9C9AF`
- `--accent-selected`: `#A5711E`

## Typography
- Title font stack: `Baskerville, Palatino Linotype, serif`
- Body font stack: `Avenir Next, Segoe UI, sans-serif`
- Title size: 30 px
- Section label size: 11 px uppercase with tracking
- Value size: 13 px tabular

## Spacing
- Outer padding: 28 px
- Internal panel radius: 28 px
- Top-row gap: 24 px
- Hero section top margin: 22 px
- Mode button gap: 12 px

## Control Styling
- Knob face uses radial gradient from pale ivory to warm cream.
- Knob ring uses gold gradient with subtle inner shadow.
- Hero knob is larger and brighter than all other controls.
- Mode buttons use cream backgrounds with gold active fill and warm shadow.

## States
- Default: quiet cream panel with gold edge detail.
- Hover: stronger gold highlight and slightly deeper shadow.
- Active: richer gold fill and brighter indicator.
- Mode change: subtitle text updates to describe the selected personality.
