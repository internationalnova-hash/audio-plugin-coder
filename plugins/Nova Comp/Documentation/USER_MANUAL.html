Mode Buttons

These buttons are fully mapped to the DSP. They change backend behavior even when the visible knobs stay in place.

MAGIC Button

MAGIC is a real DSP enhancement layer. When enabled it adds top smoothing, smarter detector behavior, slightly more gain reduction when useful, smoother release behavior, extra warmth, subtle stereo polish, and a bit more auto-trim balancing.

MAGIC does not trigger a hidden preset by itself and does not automatically move the visible knobs.

Presets

VOCAL, DRUMS, BASS, and MASTER are real recall buttons. They set the actual controls to musical starting points and may turn MAGIC on or off depending on the source.
