Nova Level User Manual

Optical Dynamics Processor

Nova Level is a simple, premium compressor built around one main control and a few meaningful buttons.

Main Controls

ControlDSP Function
COMPRESSIONLowers effective threshold and increases compression amount.
TONE SHAPINGApplies tilt EQ and changes how the detector reacts to upper content.
OUTPUTApplies final makeup gain.
GR / OUTSwitches meter view only.

Mode Buttons

SMOOTH, PUNCH, and LIMIT are fully mapped DSP modes that change ratio, attack, release, and knee behavior.

MAGIC

MAGIC adds a finishing layer of smoothing, detector polish, warmth, stereo enhancement, and auto-trim balancing.

It changes the backend sound without automatically moving the visible knobs.
