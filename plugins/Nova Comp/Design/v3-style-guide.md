# Nova Level — Style Guide v3

## Design Intent
Nova Level should be the most **self-assured** and **controlled** product in the Nova lineup.

**Key words:**
- minimal
- premium
- confident
- controlled
- polished
- iconic

## Family Positioning
- `Nova Tone` = shaping / elegant
- `Nova Heat` = energy / richness
- `Nova Level` = control / consistency / smoothness

## Layout Identity
The defining visual shape is the centered triangle:

```text
        COMPRESSION

TONE SHAPING      OUTPUT
```

This should make Nova Level immediately recognizable beside the other plugins.

## Knob Hierarchy
### Hero knob — `COMPRESSION`
- largest element in the plugin
- about `10–15%` larger than the previous version
- thicker ring
- brighter cool-gold highlight
- subtle GR feedback arc around it

### Secondary knobs — `TONE SHAPING`, `OUTPUT`
- medium size
- visually supportive
- clean and uncluttered

## Meter Language
The meter should feel more premium by behavior, not by brightness.

### Required feel
- slower falloff
- smoother movement
- slightly non-linear response
- subtle gold glow on active bars
- analog / optical, not hot / aggressive

## Tone Section Detail
To keep the lower-left area from feeling empty, add a tiny supporting cue such as:
- `Color / Presence`

This should feel informative, not busy.

## Magic Button Identity
`MAGIC` should not look like a generic button.

When ON:
- increase gold highlight
- add soft halo / bloom
- make it feel like a finishing switch

The impression should be:
> **everything just got better**

## Typography
### Title
- serif display treatment for `NOVA LEVEL`
- elegant and confident

### Descriptor
- uppercase sans-serif
- spaced lettering
- premium, technical, restrained

## UX Principle
The UI should visually reinforce:
> **Turn one main knob, get results fast, trust the sound.**
