# Nova Comp — Style Guide v2

## Design Intent
Nova Comp should feel like the most **confident and simplified** member of the Nova family.

**Key words:**
- minimal
- premium
- controlled
- iconic
- calm
- instantly usable

## Core Layout Identity
This version is defined by the **centered triangle** layout:

```text
        COMPRESSION

TONE SHAPING      OUTPUT
```

That shape is the identity.

## Knob Language
All knobs should stay inside the Nova family language:
- brushed metal cream body
- soft top-left highlight
- gold indicator line
- radial markings

### Hero knob — `COMPRESSION`
- visibly larger than the others by about `20–25%`
- slightly thicker outer ring
- slightly brighter gold accent
- optional subtle feedback arc around it to imply gain reduction

### Secondary knobs — `TONE SHAPING` and `OUTPUT`
- medium size
- balanced and supportive
- visually calmer than the hero knob

## Meter Language
- vertical analog-inspired movement
- slightly taller than Nova Heat
- slower and smoother animation
- very soft gold glow only
- should suggest "optical control" rather than hot energy

## Buttons
### Mode buttons
- compact row under the hero knob
- inactive = neutral cream
- active = soft cool-gold highlight

### Magic button
- compact premium toggle near the output area
- should feel helpful and polished, never flashy

### Preset buttons
- bottom row pills
- same Nova family geometry
- secondary to the main triangle controls

## Typography
### Main title
- serif display styling for `NOVA COMP`
- elegant, high-end, analog-inspired

### Descriptor text
- uppercase sans-serif
- lightly spaced
- reads as premium product language rather than marketing noise

## Visual Difference vs Tone / Heat
- `Nova Tone` = shaping / elegant panel layout
- `Nova Heat` = richer / more energetic panel layout
- `Nova Comp` = simpler, more centered, more iconic

This plugin should feel the most **self-assured** in the lineup.

## UX Principle
The UI should visually reinforce this message:
> **You do not need many knobs to get a polished result.**
