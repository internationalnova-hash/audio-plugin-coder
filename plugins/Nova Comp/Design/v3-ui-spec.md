# Nova Level — UI Specification v3

## Overview
**Plugin name:** `Nova Level`  
**Display title:** `NOVA LEVEL`  
**Subtitle:** `Optical Dynamics Processor`

This is the final premium direction for the compressor UI. It keeps the Nova family material system, but shifts the layout and mood so the plugin feels calmer, more controlled, and instantly recognizable.

## Core Identity
Nova Level should feel:
- minimal
- premium
- confident
- easy to trust
- fast to use

## Top Bar
### Left
- Main title: `NOVA LEVEL`
- Subtitle: `Optical Dynamics Processor`

### Right
- Branding line: `by International Nova`
- must match the suite style used in Nova Tone, Nova Heat, and Space By Nova

The message is:
> **I don’t need 10 knobs.**

## Layout Concept — Locked Flagship Layout
Nova Level should use a restrained three-part arrangement with one clear hero control:

```text
LEFT:    TONE SHAPING
CENTER:  COMPRESSION + MODES
RIGHT:   OUTPUT + METER + MAGIC
BOTTOM:  PRESETS
```

### Why it works
- top = control
- bottom-left = shaping
- bottom-right = finishing
- more memorable than another three-panel strip

## Main Controls
### 1. `COMPRESSION`
- top center
- **hero knob**
- approximately `10–15%` larger than the current draft and clearly the main focus
- slightly thicker ring and brighter highlight than the other knobs
- should include a **subtle feedback arc** to imply gain-reduction behavior

### 2. `TONE SHAPING`
- bottom-left
- medium knob
- add a small support caption such as:
  - `Color / Presence`
- should feel less empty and more intentional than before

### 3. `OUTPUT`
- bottom-right
- medium knob
- labeled clearly for level matching and finishing

## Supporting Controls
### Mode buttons
Compact row directly under `COMPRESSION`:
- `SMOOTH`
- `PUNCH`
- `LIMIT`

### Meter
- vertical on the right side
- grouped tightly under the `GR` / `OUT` toggle
- slightly taller than Nova Heat’s meter
- slower falloff and smoother animation for optical character
- active bars should have a very soft warm-gold glow

### Magic button
- bottom-right near the output section
- should feel like a finishing tool, not a normal utility button
- when ON:
  - stronger gold highlight
  - subtle halo / glow
  - feels like “secret sauce”

### Preset bar
Bottom row:
- `VOCAL`
- `BASS`
- `DRUMS`
- `MASTER`

## Color Direction
Use the same base champagne system, but make the accent more **cool gold / neutral champagne** than Nova Heat.

### Palette
- **Faceplate:** `#F0E4D2`
- **Faceplate shade:** `#E3D6C0`
- **Inner panel:** `#D8C9AE`
- **Dark text:** `#453C34`
- **Soft text:** `#6B655D`
- **Cool gold:** `#B8924E`
- **Deep gold:** `#8F713B`
- **Pale highlight:** `#EAD7A6`
- **Cream white:** `#FFF9F1`

## Premium Details
1. **Compression feedback arc** around the hero knob
2. **Slower meter falloff** for optical feel
3. **Subtle active-bar glow** on the meter
4. **Magic halo** when enabled
5. **Small tone subtitle** to strengthen the lower-left section

## UX Flow
A user should naturally do this:
1. turn `COMPRESSION`
2. adjust `TONE SHAPING`
3. match with `OUTPUT`
4. tap `MAGIC` if they want the sound to feel more finished

The result should feel like:
> **A modern LA-2A-inspired compressor for creators who don’t want to think.**
