# Nova Comp — Style Guide v1

## Design Intent
Nova Comp should feel like the **control** unit of the Nova family.

**Key words:**
- refined
- controlled
- premium
- analog
- balanced
- effortless

## Palette
| Role | Hex | Notes |
|------|-----|-------|
| Main faceplate | `#EFE1C7` | cool champagne / cream base |
| Faceplate shadow | `#E3D2B4` | gentle depth without warmth overload |
| Inner panel | `#D8C5A0` | slightly darker hardware inset |
| Dark text | `#473D33` | strong label contrast |
| Soft text | `#6B6258` | supporting copy |
| Cool gold | `#BE9440` | main accent |
| Deep gold | `#9E7330` | active states and emphasis |
| Pale gold | `#E8CC86` | highlights, knob arcs |
| Glass cream | `#FFF9F1` | reflections and readouts |
| Meter olive-gold | `#B7A66A` | calmer analog meter tone |

## Typography
### Title
- Serif display type for `NOVA COMP`
- premium mastering-hardware feeling
- strong but not aggressive

### Section labels and captions
- Sans-serif uppercase
- spaced lettering
- clear, understated, technical elegance

### Readouts
- compact pill-style numeric displays
- bold enough for quick scanning

## Knob Language
- same Nova family metal-knob system
- brushed cream-metal body
- gold indicator line
- subtle radial markings
- `LEVEL` should be visually largest and most inviting
- `GAIN` should feel solid and trustworthy, not flashy

## Meter Language
Unlike Nova Heat, the compressor meter should feel:
- slower
- smoother
- more analog
- less intense in color

The meter should imply classic hardware movement rather than hot saturation energy.

## Button Styling
### Mode buttons
- small pill buttons
- inactive = neutral cream
- active = soft gold fill with subtle shadow

### Magic button
- compact premium toggle
- active state should glow softly, not dramatically
- should feel like “extra polish,” not excitement

### Preset buttons
- bottom row pills matching the Nova family
- consistent spacing and rounded geometry
- clear but secondary to the main compression controls

## Layout Rules
- generous side padding
- clear left / center / right functional grouping
- avoid crowding the output section
- make the meter and gain relationship obvious
- preserve a quick-read hardware workflow

## Family Positioning
- `Nova Space` = airy / deep
- `Nova Tone` = elegant / shaping
- `Nova Heat` = energetic / rich
- `Nova Comp` = controlled / smooth / polished

Nova Comp should therefore be the most **calm and confidence-building** member of the lineup.
